DUNGEON.txt

A modern text adventure made for a DND based game jam. I don't know a lot about dnd, so I made most of it up, but it ended up being one of my favorite projects so far.


Everything from the characters, to the backgrounds, to the UI, is made using text symbols in an Ascii/Petsii style.


On thing im particularly proud about the UI that slides in and out of frame when needed, adjusting all the other UI elements with it.
I also use a lot of little tricks to make the game feel like it had more content than it did, like mazes, and finding items hidden around the area.
